# MINPI — Minimal Perspective Interface

MINPI est une API graphique ultra‑légère conçue pour afficher des mondes en blocs
avec une perspective propre, stable et extrêmement rapide.  
Elle offre une alternative minimaliste à OpenGL pour les jeux voxel / Minecraft‑like,
en simplifiant le pipeline GPU tout en maximisant les performances.

---

## 🚀 Pourquoi MINPI ?

- Ultra simple à utiliser  
- Optimisée pour les petits GPU  
- Pipeline minimal : quads + atlas + perspective  
- Pas de shaders complexes  
- Pas de moteur 3D lourd  
- Parfait pour les jeux de blocs, simulations, prototypes et moteurs custom

- une caméra en perspective
